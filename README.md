📝 TaskManager - Full Stack Spring Boot Application
A robust Task Management system built with Spring Boot, focusing on secure authentication and scalable architecture. This project demonstrates professional Java development standards, including stateless authentication and a strictly decoupled layered architecture.

🚀 Key Features
JWT Authentication: Secure user sessions using JSON Web Tokens and Spring Security.

RESTful API: Clean, resource-oriented endpoints for task and user management.

Layered Architecture: Follows industry standards for maintainability:

Controller Layer: Manages HTTP requests and API documentation.

Service Layer: Handles core business logic and transaction management.

Repository Layer: Performs database operations via Spring Data JPA.

Entity Layer: Defines the schema for the PostgreSQL database.

Data Security: Password encryption using BCrypt and protected API routes.

🛠️ Tech Stack
Language: Java 17+

Framework: Spring Boot 3.x

Security: Spring Security, JWT

Database: PostgreSQL

ORM: Spring Data JPA / Hibernate

Build Tool: Maven
